# Web Tools Lab

This web tools lab repository is a collection of lightweight web-based utilities designed to help developers, designers, and engineers work faster.

These tools are part of the Rooted Dreams developer resources initiative.

## Tools Included

- JSON Formatter & Validator
- Secure Password Generator
- Base64 Encoder / Decoder
- Regex Pattern Tester
- Color Palette Generator

## Purpose

The goal of this project is to provide simple, fast developer tools that can be used directly in the browser without installing any software.

These tools are especially useful for:

- Web developers
- API developers
- UI/UX designers
- Software engineers
- Students learning programming
