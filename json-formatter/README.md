# JSON Formatter & Validator

A free, browser-based JSON formatter, validator, and beautifier — no server, no installation, no sign-up.

**🔗 Live tool:** [rooteddreams.net/tools/json-formatter](https://www.rooteddreams.net/tools/json-formatter)

## Features

- Instant JSON formatting with syntax highlighting
- JSON validation with clear error messages
- Minify / compress JSON output
- Configurable indentation (2 spaces, 4 spaces, or tab)
- Key count and byte size display
- Auto-format on paste
- Copy to clipboard
- Runs 100% in the browser — no data is sent to any server

## Usage

Open `index.html` in any browser. No build step or dependencies required.

```
json-formatter/
├── index.html
├── css/
│   └── style.css
└── js/
    └── app.js
```
